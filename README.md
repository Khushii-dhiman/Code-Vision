# 🧠 Code-Vision

Code-Vision is a **Data Structures & Algorithms Visualizer** built using **React.js + Vite + Tailwind CSS**.  
It helps users understand how sorting algorithms work step-by-step through interactive animations.

---

## 🚀 Features

- 📊 Bubble Sort Visualization
- 🎚️ Adjustable array size
- ⚡ Speed control for animation
- 🔄 Reset / regenerate array
- 🎯 Interactive UI with smooth animations
- 📱 Responsive design

---

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- Tailwind CSS
- HTML/CSS
